
# 실행방법

```bash
# Start the Metro Server
npm start

# Start Android App 
npm run android

# Start IOS App
cd ios 
pod install
cd ..
npm run ios
```

# 실행 환경 
> node: v23.5.0  
> cocoapod: 1.15.2
